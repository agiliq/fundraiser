=======================
How to setup fundraiser
=======================

Steps
-----
    1.Clone `fundraiser <https://github.com/agiliq/fundraiser.git>`_. repository into your project directory
        git clone https://github.com/agiliq/fundraiser.git
    2.Make sure you have installed all the depedencies
        pip install -r requirements/base.txt
    3.You are all set to go now.
