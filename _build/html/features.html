Create a campaign¶

Beneficiaries can start campaigns by entering the following details Campaign name, Target amount, Cause, Image.

Viewing details of campaigns¶

All can view details of any running campaigns.

Donating to a campaigns¶

Donors can donate to any campaigns using their Credit/Debit details
